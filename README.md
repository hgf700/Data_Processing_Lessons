# Lessons About Data

The goal of these lessons is to introduce fundamental concepts and practical skills related to data analysis, visualization, preprocessing, and machine learning.

## Lab 1 — Introduction to Pandas and Data Transformation
- Working with the Pandas library
- Loading and exploring datasets
- Data transformation and manipulation
- Basic statistical operations

## Lab 2 — Data Visualization
- Visual exploration of datasets
- Using visualization libraries

## Lab 3 — Data Cleaning
- Handling missing values
- Removing duplicates
- Data normalization and formatting

## Lab 4 — Introduction to R and Basic Machine Learning
- Basics of the R programming language
- Simple classification models
- Simple regression models
- Model evaluation metrics
- Introduction to Ridge and Lasso regression

## Lab 5 — Supervised Learning
- Classification techniques
- Regression techniques
- Training and testing models
- Performance evaluation

## Lab 6 — Unsupervised Learning
- Clustering methods
- Dimensionality reduction
- Pattern discovery in datasets
- Comparing supervised and unsupervised approaches

## Lab 7 — Time series data analysis
- Moving averages (7-day and 30-day windows)
- Analysis of ACF and PACF (Autocorrelation and Partial Autocorrelation functions)
- Decomposition of time series into:
  - Trend
  - Seasonality
  - Residual components
